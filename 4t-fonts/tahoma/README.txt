This font, Tahoma, is included with all major operating systems, including virtually all Microsoft Windows
operating systems and Apple Mac OS X. If you use one of these operating systems, you do not need to manually
install this font.

However, in the event you do not have this font installed or use a different operating system, the font is
included with the free Microsoft Word 97 Viewer, which can be downloaded from the following URL:

http://prdownloads.sourceforge.net/corefonts/wd97vwr32.exe?download
